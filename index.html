Highlights

Tabbed editor

Line numbers and current‑line highlight for quick orientation.

Syntax highlighting

Essentials for Python, JSON, and Markdown.

File explorer dock

With fixed Windows path handling for painless navigation.

Command palette

Fuzzy filtering via Ctrl+Shift+P for everything.

Project‑wide search

Threaded search that keeps the UI smooth while it scans.

Integrated task runner

Run arbitrary shell commands without leaving the editor.

Git panel

Status, stage‑all, commit — powered by your local git CLI.

Markdown preview dock

See your docs render live alongside the editor.

Persistent settings

Saved to ~/.pycodex_pro/settings/settings.json.

Tiny plugin system

Drop .py files into .../plugins/user to extend.

main.py — PyCodeX Pro 
# PyCodeX Pro — quick taste
from pycodex import Editor
app = Editor()
app.open("README.md")
app.run("build")  # task runner
app.git.commit("feat: first commit")
app.show_markdown_preview(True)

About

PyCodeX Pro is a compact, VS Code–style editor built with PySide6. It focuses on look, responsiveness, and the core features most people expect from VS Code—without bloat.
